# WhatsHapp [![Build Status](https://travis-ci.org/tchu88/whats-happ.svg?branch=master)](https://travis-ci.org/tchu88/whats-happ)

## Setup

```console
git clone https://github.com/tchu88/whats-happ.git
cd whats-happ
bundle install
rake db:create
rake db:migrate
rake db:test:prepare
rake # run the tests
rails server
```
